IndLish - An Indian language reader

This extension detects and converts Indian language script into readable English.

It will be useful for the people, who can understand the language but can't read/write.

To read a text in English, Select the text and click on IndLish Covert context menu.


Supported Languages:
  • Bengali
  • Hindi
  • Kannada
  • Malayalam
  • Marata
  • Tamil
  • Telugu
